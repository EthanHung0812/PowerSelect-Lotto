威力彩選號與兌獎系統

開發環境：
平台：Windows Forms
開發語言：C#
開發工具：Visual Studio

功能介紹：
本系統為一套全面的威力彩選號與兌獎應用程式，設計用於增加使用者參與樂透的互動性與便利性。

系統主要功能包括：
開獎：模擬威力彩開獎過程，隨機生成一組中獎號碼。
自動選號：提供使用者隨機選擇號碼的功能，可生成單組或多組號碼。
手動選號：使用者可手動選擇喜好的號碼。
隨機產生n組選號：根據使用者需求，一次性隨機生成多組號碼。
加入選號：將選好的號碼加入到系統中，用於後續的兌獎。
兌獎：與系統中的開獎號碼進行匹配，確認中獎情況。
顯示中獎紀錄：顯示所有中獎記錄及相關細節。
計算中獎金額：根據中獎的類型計算總中獎金額。

系統特點：
簡潔介面：使用簡潔直觀的用戶介面，方便所有年齡層的用戶操作。
靈活性：支持多種選號方式，滿足不同用戶的需求。
即時反饋：兌獎結果立即顯示，增加使用者的參與感。

適用對象：
威力彩愛好者
想要練習選號技巧的使用者
本系統透過模擬真實的威力彩環境，提供一個有趣且互動的平台，讓使用者能夠在無風險的情況下體驗選號和兌獎的過程。
